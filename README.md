# student-management-system-python
A simple console-based Student Management System built in Python that allows users to add, view, search, update, and delete student records.

## Features
- Add a new student
- View all students
- Search student by ID
- Update student marks
- Delete a student

## How to Run
1. Make sure Python 3 is installed
2. Run the program using:

```bash
python student_management_system.py
